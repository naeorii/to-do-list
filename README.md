Projeto de To Do List (lista de tarefas) utilizando HTML, CSS e JavaScript, com foco na prática de manipulação do DOM e lógica de programação. O sistema permite adicionar, marcar como concluídas e remover tarefas, além de salvar automaticamente os dados no localStorage, garantindo que as informações permaneçam mesmo após recarregar a página. O projeto também explora conceitos de estilização responsiva, organização de código e interação com o usuário, contribuindo para o desenvolvimento de habilidades em front-end e desenvolvimento web.
